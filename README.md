# ⚔️ VimQuest

**Learn Vim from baby steps to mastery** — interactive lessons, live practice files, and games, all inside VS Code.

## Features

- 📚 **18 structured lessons** across 4 chapters: Baby Steps → Moving Like a Pro → Editing Powers → Intermediate → Advanced
- 📝 **Live practice files** — every lesson opens a scratch file so you can try commands immediately
- 🎮 **3 games**: Command Quiz, Sequence Builder, and Flashcard Drill
- ⚡ **XP & rank system** — earn XP as you complete lessons
- 🔥 **Daily streak tracking**
- 🔒 **Progressive unlock** — complete each lesson to unlock the next

## Getting Started

1. Click the **VimQuest sword icon** in the Activity Bar (left sidebar)
2. The Welcome screen opens automatically on first launch
3. Click **Start Learning** or pick any lesson from the sidebar
4. For live practice: also install the **[Vim extension by vscodevim](https://marketplace.visualstudio.com/items?itemName=vscodevim.vim)**

## Lessons

### 🌱 Baby Steps
- What is Vim? Modes 101
- Move without arrow keys: h j k l
- Entering INSERT mode: i a o
- Save & Quit: :w :q :wq

### ⚡ Moving Like a Pro
- Word jumps: w b e
- Line starts & ends: 0 ^ $
- File navigation: gg G {line}G
- Find on line: f t F T

### 🔥 Editing Powers
- Delete, change, yank: d c y
- Undo & Redo: u Ctrl+r
- Visual mode: v V Ctrl+v
- Text objects: iw aw ip i" a(

### 🚀 Intermediate
- Marks: m{a-z} '{a-z}
- Registers: named registers
- Macros: record and replay

### 🏆 Advanced
- Search & Replace: :s and :%s
- Splits & Tabs: :sp :vsp :tabnew
- The dot command: repeat last change

## Games

- **⚡ Command Quiz** — 10 questions, multiple choice, timed rounds
- **🧩 Sequence Builder** — type the correct Vim command for a given task
- **📋 Flashcard Drill** — flip cards to reveal command meanings

## Requirements

- VS Code 1.80+
- (Recommended) [Vim extension by vscodevim](https://marketplace.visualstudio.com/items?itemName=vscodevim.vim) for live practice

## Tips

- Use the **"Open Practice File"** button in every lesson to try commands on a real file
- Enable relative line numbers: `"editor.lineNumbers": "relative"` in VS Code settings
- Play the games after finishing each chapter to reinforce what you learned
