# ⚔️ VimQuest

**Learn Vim step by step** — from baby steps to advanced mastery, with interactive lessons, live practice files, and games. All inside VS Code.

[![License: MIT](https://img.shields.io/badge/License-MIT-purple.svg)](https://opensource.org/licenses/MIT)

## ✨ Features

- 📚 **18 structured lessons** across 5 chapters (Baby Steps → Moving → Editing → Intermediate → Advanced)
- 📝 **Live practice files** — every lesson creates a scratch file so you can try commands live
- 🎮 **3 games**: Command Quiz (50 questions), Sequence Builder, Flashcard Drill
- ⚡ **XP & rank system** — earn XP completing lessons, level up your rank
- 🔥 **Daily streak tracking**
- 🔒 **Progressive unlock** — complete each lesson to unlock the next

## 🚀 Getting Started

1. Click the **⚔️ VimQuest sword icon** in the VS Code Activity Bar (left sidebar)
2. The Dashboard opens automatically on first install
3. Click **Start Learning** or pick any available lesson from the sidebar
4. For live practice: also install **[Vim by vscodevim](https://marketplace.visualstudio.com/items?itemName=vscodevim.vim)**
